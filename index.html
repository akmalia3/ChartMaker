About This Application


This application will visualize your data into a graphics by a manual input from user. This application build with html, css and javascript. To make a graphics we use chartjs library and use electron to build into an application destkop.


To use this application, you just want to input a number in input column that available in page and the chart will change by dynamic changes. For now you can't save your data, but this still in progress. So don't worry about it.
